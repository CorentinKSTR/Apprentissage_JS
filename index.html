Le QuerySelectAll !!!

  • This is a
  • nav ul li
  • with a CSS Hover

Le QuerySelectorAll nous permet de selectionner n'importe quel groupe d'éléments (dans notre cas des balises "li"),
pour leurs donner des instructions à l'aide d'un "for each". Ici, nous cherchons à créer un effet hover sur les "ul li" de la nav bar.

var li = document.querySelectorAll("li")li.forEach((e)=>{
            e.addEventListener('mouseover', f => {
            e.classList.toggle("li-hover")
            })
        })
        li.forEach((e)=>{
            e.addEventListener('mouseout', f => {
            e.classList.toggle("li-hover")
            })
        })

Le Dark Mode !!!

La fonction addEventListener "onclick" permet de créer un évennement après un clique sur le bouton "dark mode". Une div (id="darkmode") est crée sous le body, pour nous permettre de modifier le style de l'ensemble du document. La fonction "classList" permet de créer une nouvelle class dans la div darkmode (html) cette class est également assignée en CSS, cette nouvelle class sera supprimée de la div avec la fonction "toggle" après un nouveau clique sur le bouton Dark Mode. 

var darkmode = document.getElementById("darkmode") 
var darkbutton = document.getElementById("darkbutton")
darkbutton.addEventListener('click', e=>{
    darkmode.classList.toggle("darkmode")
})

Une image qui change de comportement au survolage de la souris

Html 


                <div id="idimg">
                    <img id="imagehover" src="img/jsimage.png">
                </div>

CSS 


            #imagehover{
                -webkit-filter: grayscale(100%);
                margin-top: 5%;
                width: 15%;
                height: 15%;
                transition: width 1s, height 1s, transform 1s;
            }
            
            #imagehover:hover{
                -webkit-filter: grayscale(0%);
                width: 25%;
                height: 25%;
                transform: rotate(360deg);
            }

Explication 


                Cette animation n'a pas besoins de javascript, 
                tout se passe en CSS lorsqu'on passe la souris sur l'element avec l'atribu ":hover"
                "-webkit-filter: grayscale(100%);" permet de mettre l'image en noir et blanc
                "-webkit-filter: grayscale(0%);" permet de la remettre en couleur
                "transition: width 1s, height 1s, transform 1s;" permet de gerer le délais de l'animation (taille, rotation)

Animer la couleur d'une div

Refresh the page

html

<div class="losange"></div>

css

Le transform rotate sert à le changerr de position, c'est propriété est indépendant de l'animation car ce n'est pas une transition. Ici le keyframe losange_changing_colors est l'animation qui est appelée juste au dessus et chaque pourcentage s'applique à la durée qui est de 4s.
Donc à 25% de 4s la couleur de la div devient lightyellow, au bout de 2s elle devient lightgreen ainsi de suite...
Info utile à savoir au bout des 100% elle affiche white pour ensuite elle affiche la couleur initiale ici white pour que l'effet soit plus sympas mais dans l'hypothèse ou la couleur initiale est autre que white la transition se ferrai sans delai.


                .losange{
                    margin: 60px;
                    background-color: white;
                    height: 260px;
                    width: 260px;
                    transform: rotate(45deg);
                    animation-name: losange_changing_colors;
                    animation-duration: 4s;
                }
                @keyframes losange_changing_colors {
                    0%  {background-color: lightgreen;}
                    25%  {background-color: lightyellow;}
                    50%  {background-color: lightgreen;}
                    75%  {background-color: lightskyblue;}
                    100% {background-color: white;}
                }

Faire une fonction JavaScript qui change l'image

html

J'ai mis dans ma balise onclick="change_img()" qui déclenche la fonction de mon fichier js qui est nommée change_img. 

 
                <p><img class="img_lightgreen" id="img_lightgreen" src="img/img_lightgreen.jpg" alt="image with green abstrait art" onclick="change_img()">
            <img class="img_lightpink" id="img_lightpink" src="img/img_lightpink.png" alt="image with pink abstrait art" onclick="change_img_back()"></p>

css


            .img_lightgreen, .img_lightpink{
                height: 500px;
                width: 700px;
            }
            .img_lightpink{
                display: none;
            }

js

Je créer d'abord deux variables qui vont chercher dans mon fichier html les éléments qui ont pour id respectifs img_lightpink et img_lightgreen.
je créer une fonction change_img() qui va ajouter une balise style dans mon fichier html dans sa balise initiale.
Donc je change le display de l'image que je veux cacher en none et le display de celle que je veux afficher en block. 

 
                var img_lightpink = document.getElementById("img_lightpink")
                var img_lightgreen = document.getElementById("img_lightgreen")
                
                function change_img(){
                    img_lightpink.style.display = "block"
                    img_lightgreen.style.display = "none"
                }
                function change_img_back(){
                    img_lightpink.style.display = "none"
                    img_lightgreen.style.display = "block"
                }
