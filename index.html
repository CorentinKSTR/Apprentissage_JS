getElementById()

La méthode getElementById() de Document renvoie un objet Element représentant l'élément dont la propriété id correspond à la chaîne de caractères spécifiée.

Exemple

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus , eum ex possimus nulla, debitis porro fugiat alias laborum mollitia unde tempore illo at nam facere! Id maiores necessitatibus voluptatum at!

HTML
<div class="txt_lorem">
        <h3>Exemple</h3>
        <p id="if">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus
        , eum ex possimus nulla, debitis porro fugiat alias laborum mollitia unde tempore illo at nam facere! Id maiores necessitatibus voluptatum at!</p>
        <button onclick="Button2()">getElementById</button>
        </div>
CSS
.txt_lorem h3{
        padding-bottom: 10px;
    }
    .txt_lorem, .txt_lorem2 {
        border: 2px inset black;
        text-align: center;
        width: 900px;
        height: auto;
        margin: auto;
        margin-top: 100px;
        padding: 10px;
        background-color: #c2f8cb;
    }
    .txt_lorem button{
        margin-top: 20px;
    }
    .txt_lorem:hover{
        box-shadow: 2px 2px 4px black;
    }
JS
function Button2() {
        document.getElementById("if").innerHTML = "Lorem ipsum";
      }

getElementByClassName()

La méthode getElementByClassName() est similaire à la fonction mise un peu plus haut, juste que ici, on utilise une Class et non un ID.

Exemple

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus , eum ex possimus nulla, debitis porro fugiat alias laborum mollitia unde tempore illo at nam facere! Id maiores necessitatibus voluptatum at!

HTML
<h3>Exemple</h3>
                <p class="p_OnEstLa">Lorem ipsum dolor sit amet, consectetur adipisicing elit. Delectus
                , eum ex possimus nulla, debitis porro fugiat alias laborum mollitia unde tempore illo at nam facere! Id maiores necessitatibus voluptatum at!
                <button onclick="Button3()">getElementByClassName</button>
                </div>
CSS
.txt_lorem h3{
                padding-bottom: 10px;
            }
            .txt_lorem, .txt_lorem2 {
                border: 2px inset black;
                text-align: center;
                width: 900px;
                height: auto;
                margin: auto;
                margin-top: 100px;
                padding: 10px;
                background-color: #c2f8cb;
            }
            .txt_lorem button{
                margin-top: 20px;
            }
            .txt_lorem:hover{
                box-shadow: 2px 2px 4px black;
            }
JS
function Button3() {
                var x = document.getElementsByClassName("p_OnEstLa");
                x[0].innerHTML = "Hello World!";
              }

querySelector()

La méthode querySelector() de l'interface Document retourne le premier Element dans le document correspondant au sélecteur.

Exemple

HTML
<div class="example">
                        <h1>Exemple</h1>
                        <button class="btn_title" onclick="Button3()">Try it</button>
                        </div>
CSS
.example{
                        text-align: center;
                        border: black inset 2px;
                        margin-top: 50px;
                        background-color: #c2f8cb;
                        width: 900px;
                        height: auto;
                        margin-right: auto;
                        margin-left: auto;
                    }
.example button{
                        margin-top: 10px;
                    }
JS
function Button3() {
                        document.querySelector(".example").style.backgroundColor = "red";
                      }
