Un bouton qui fait pivoter un element au clic

Apprentissage JS

Html 


        <h1 id="title">
            Apprentissage JS
        </h1>
        <div class="content">
            <div>
                <button class="anim" onclick="Button()">
                    Animation
                </button>
            </div>
        </div>

CSS 


            .animation{
                transform: rotate(180deg)!important;
                transition: transform 0.1s;
            }

javascript 


            function Button(){
                document.getElementById("title").classList.toggle("animation")
              }

Explication
